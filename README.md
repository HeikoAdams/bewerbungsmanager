# Bewerbungsmanager
Der Bewerbungsmanager ist eine kleine, in FreePascal/Lazarus geschriebene Anwendung
zur Verwaltung von Bewerbungen.

Bewerbungsmanager bietet unter anderem folgende Funktionen:
